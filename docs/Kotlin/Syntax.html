    # When

    Kotlinは、ifも使えるけど`whenで似たような使い方もできる

    # 引数がある場合

    whenの後ろに引数をつけるとその変数から条件分岐するようになる

    それぞれの条件分岐は 条件 -> {条件分岐したら実行する処理}で書く

    条件でelseを使用するとifと同じelseと同じ動きをする

    条件は,で区切ると||演算子のような処理になる

    val value = 2

when (value) {
 0 -> {
   println("0")
 }
 1, 2 -> {
  	println("1 || 2")
 else -> {
  	println("else") 
 }

    # 引数がない場合

    whenに引数をなしにすることもできる

    条件分岐の書き方は同じく 条件 -> {条件分岐したら実行する処理}書く

    条件には||と&&演算子も使うことができる

    val value1 = 2
val value2 = 3

when (value) {
 value1 == 0 -> {
   println("0")
 }
 value1 == 1||value2 ==2 -> {
  	println("1 || 2")
 else -> {
  	println("else") 
 }
