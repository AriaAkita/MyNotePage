    # Ubuntuでhome bridgeを使う方法

    Node.js,HomeKit,Linux

    UbuntuでHome bridgeを使った時少しだけ他のOSと違かったから書いておくよ

    # home bridgeをinstallする

    home bridgeを次のCommandでinstallするよ

    homebridge-config-ui-xを使うと、Web Serverを立ち上げてHome bridgeを画面上から操作できるよ

    $ sudo npm install -g --unsafe-perm homebridge homebridge-config-ui-x

    # home bridgeの実行とGUI付きの画面を立ち上げる

    次のCommandでhome brigadeとWebServerの立ち上げを行うよ

    $  sudo hb-service install --user ${実行するUser名}

    WARNING

    実行するUser名は、Linuxだと必須なので気をつけること！

    これでhome bridgeが使えちゃう

    細かい設定は画面でやってね
