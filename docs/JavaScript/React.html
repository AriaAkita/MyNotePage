    # Hooksってなに？

    Hookとは、 React の機能に「接続 (hook into)」するための特別な関数のこと。

    # State Hook

    今までのstateのやり方だと次のような感じに書いたけど、

    import React from 'react';

class Man extends React.Component {
    constructor(args) {
        super(args);
        this.state = {
            age: 10   
        };
    }
}

    Hookを使うとこんな感じにできる。

    import React, { useState } from 'react';

function Man() {
  // 変数名 と setterをつける
  const [age, setAge] = useState(10);
}
