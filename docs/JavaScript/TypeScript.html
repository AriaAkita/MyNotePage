    # TypeScript

    # TypeScriptのType Alias

    # Type Aliasってなに？

    いろいろなところで使いたいType Annotationの名前を使うための構文だよ

    型や組み合わせに別の名前をつけるために使うよ

    # 使い方

    typeをつけてその後に変数名をつけて型アノテーションを代入すれば使えるよ

    interfaceとは違って、どんな型アノテーションにも与えることができるよ

    // 文字列か数字の型なことがわかるよ
type StringOrNumber = string|number;

let sample: StringOrNumber
sample = 12
sample = 'name'

type Text = string | { text: string }
type Array = [number, string|number|boolean]
type Callback = (arg: any) => void

    TIP

    Type Annotationが、階層構造をもつ必要があるときはinterfaceを使うよ

    # TypeScriptのInterface

    # Interfaceってなに？

    Interfaceは、Object、Classの規格を定義するよ

    主に抽象な型を定義するのに使うよ

    # 使い方

    interfaceを先に書いてからInterface名を書いて{}で定義を書くよ

    interface Point {
 	x: number,
    y: number
}

interface Size {
	width: number;
    heigt: number;
}

let boxSize : Size = {
    width: 8,
    heigt: 12
}

    # TypeScriptで画像をimportする方法

    JavaScript,Node,TypeScript

    importした時に何もしないと画像を読み込んでくれないけど設定をすればできるようになるよ

    # *.d.jsを作る

    まず画像のための型定義ファイルを作成するよ

    画像はTypeScriptで定義されていないので、moduleとして宣言するよ

    次のような感じで必要な拡張子を設定してね

    declare module "*.jpg"
declare module "*.svg"
declare module "*.ico"
declare module "*.png"

    WARNING

    流石にないと思うけど、Projectフォルダ内におくようにすること

    # tsconfig.jsonを設定する

    tsconfig.jsonで、型定義したファイルをincludeにファイルを読み込むように設定するよ

    {
  "compilerOptions": {
    ...
  },
  "include": [
    "src/type/**"
  ]
}
