    # よく使いそうな ip コマンド

    Shell,Cli

    ifconfigが使えなくて困った時にipで変わりができることがわかったのでまとめた

    # IPアドレスを調べる方法

    IPアドレス以外のInterfaceの情報も調べられる

    // 全部のInterfaceの情報か個別のInterfaceの情報
$ ip addr show {interface | なくてもいいよ}
$ ip a show {interface | なくてもいいよ}
// 全部のInterfaceの情報
$ ip addr 
$ ip a

    # Interfaceを起動する方法

    Wifiとかを起動するときに使うcommandだよ

    $ ip link set {interface name} up

    # Interfaceを停止する方法

    起動しているWifiとかを停止するときに使うcommandだよ

    $ ip link set {interface name} down

    # 経路情報を表示する

    経路情報を表示したい場合もipコマンドを使ってできる

    $ ip route
$ ip r

    # Disk容量の確認するコマンド

    # dfコマンド

    ファイルシステムのディスク容量を表示するコマンド

    No space left on deviceってエラーが出たときに使った

    # df

    $ df {option} {directory}

    # オプション

    オプション 説明
    -a,--all Dummy Fileも含めて容量が0のDiscも表示する
    -h,--si 単位をつけて表示する

    # 使ってるOSやProcessorの種類を見たい

    # unameコマンド

    unameコマンドを使うとOSの名前やOSから使っているProcessorの種類がわかるよ

    # OSの種類を見たい

    $ uname
$ uname -s
    # Linuxでやってみた
    $ uname
Linux

    # OSのRelease Version を見たい

    $ uname -r
    # Linuxでやってみた
    $ uname -r
4.19.66-v7+

    # OSのVersionの情報を見たい

    $ uname -v
    # Macでやってみた
    Darwin Kernel Version 19.4.0: Wed Mar  4 22:28:40 PST 2020; root:xnu-6153.101.6~15/RELEASE_X86_64

    # Hardware nameを見たい

    $ uname -m
    # Raspberry PIでやってみた
    $ uname -m
arm7v

    # Processor の Architecture を見たい

    $ uname -p
    # Raspberry PIでやってみた
    $ uname -p
i386

    # 使っている端末の Node nameを見たい

    $ uname -n

    # 上記をまとめて全て見たい

    $ uname -a

    # MakefileのTask Target

    # PHONY

    .PHONY は、Target Fileではなくて、Task Targetにするときに使用するよ

    .PHONYを使わなくてもできるけど、Task Targetと同じファイル名があったときは使わないといけない

    意味的には偽りのTargetを定義している意味になるよ

    .PHONY clean
clean:
	rm file

    # hostnameをcommand使わずに調べる方法

    間違えてhostnameでhostnameを変えてしまったので、変えないで確認する方法を書くよ

    unameのoptionにnをつけるとhostnameを表示することができるよ

    $ uname -n
