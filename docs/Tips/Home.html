    # Webを支える技術を読んで

    Web、book

    # 様々なWebの用途

    Webは同じ技術を使っていても用途はたくさんあるよ

    その中で次の3つをあげるよ

    • Webサイト
    • ユーザインタフェースのWeb
    • プログラム用APIのWeb

    # Webサイト

    Webサイトは、Webページ（html）の集合のこと

    企業の情報やショッピング、ブログ、ソーシャルネットワーク、Wikiなどのサービスを提供するWebサイトがあるよ

    Webサイトの代替のシステム構成が、PCをベースにしたUNIXやWindows、LinuxのOSのケースを数千・数万の組み合わせたサーバを組み合わせたものとかたくさんある

    Webサイトの特徴に、Webサイトの裏側がどんな構成になっているかクライアントが意識しないで済むことがある

    ユーザが難しいことを考えずにサービスを使えることはいいこと

    Webサイトはいろんなデバイスで見ることができる

    例えば、携帯電話からゲーム機、テレビでも見れる

    しかもNINTENDO Switchのメニュー画面はReactで動いている

    # ユーザインタフェースのWeb

    ルータとかプリンタの設定画面がWebブラウザから観れるけどあれのこと

    WindowsのHelpとかもHTMLの書いてあるので含まれるっぽい

    もしかするとNINTENDO Switchはこっちに含まれる気がした

    # プログラム用APIのWeb

    APIは、Application Programming Interfaceの略で、人向けではなくプログラム向けのインタフェース

    プログラム用なのでデータフォーマットには、人が読めないもの使ってる

    例えば、XML（Extensible Markup Language）とかJson（JavaScript Object Notation）とかのプログラムがわかりやすいものを送ってるよ
